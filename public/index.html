Projects

CLIMAT3

While working for Dr. Karl Maier at TIESInstitute, I led a team of four to create an application that utilized BeautifulSoup to scrape data from PubMed and other scholarly article sites. The scraped articles were then classified using the SVM algorithm and uploaded to a Cloud Firestore database. This data was visualized on the CLIMAT3 website, built through Firebase and accessed via Node.js interactions. As the team leader, I ensured project milestones were met, facilitated communication between the team and the client, and developed the entire backend of the website. The website can be found at climat3.web.app.
